# Changelog

All notable changes to the Global Smart TV / Connected TV Market Dashboard will be documented in this file.

本ファイルでは、Global Smart TV / Connected TV Market Dashboard の主な更新履歴を記録します。

## [1.0.0] - 2026-07-06

### Added

- Initial public version of the Global Smart TV / Connected TV Market Dashboard.
- Added interactive dashboard covering the 2015–2025 global Smart TV / Connected TV market.
- Added visualization of major Smart TV manufacturer shipment trends.
- Added visualization of TV OS / Connected TV platform reach.
- Added regional connected-TV OS share views for North America, Europe, and Japan.
- Added source filtering linked to dashboard tabs.
- Added evidence links to official sources, public reports, research references, and archived URLs where available.

### Documentation

- Added bilingual README summary in English and Japanese.
- Added `DATA_SOURCES.md` to summarize source categories and evidence structure.
- Added `METHODOLOGY.md` to explain data definitions, assumptions, limitations, and analytical scope.

### Notes

- Some figures are based on official disclosures.
- Some figures are based on market research estimates.
- Some regional OS share figures may include analyst-derived assumptions where direct public disclosure is unavailable.
- Measurement definitions may differ across sources, including shipments, active devices, installed base, monthly active users, households, and cumulative device reach.

## Future Update Candidates
